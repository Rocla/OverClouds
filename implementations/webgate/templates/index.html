This page has been automatically generated because you didn't add a "index.html" file with your shared files. A single file would also produce this page.

The reason for this file is that every shared on Overclouds are considered as a webapps.

You can either edit "index.html" or simply call your file by adding its name to the url path

For example:
If your file was named "movie.mp4", and your url path "https://overclouds.ch/goto/abcde..ghij/". You need to add the file name to end of your url path.
The result would be "https://overclouds.ch/goto/abcde..ghij/movie.mp4".
